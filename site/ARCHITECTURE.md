# Архитектура сайта MystVPN

## Общая архитектура

Сайт построен по принципу **Single Page Application (SPA)** с модульной структурой.

```
┌─────────────────────────────────────────────────────────────┐
│                        index.html                           │
│  ┌─────────────┐  ┌─────────────┐  ┌──────────────────┐  │
│  │    <head>   │  │   <style>   │  │     <script>     │  │
│  │  Метаданные │  │   Стили     │  │   i18n + логика   │  │
│  └─────────────┘  └─────────────┘  └──────────────────┘  │
│  ┌───────────────────────────────────────────────────────┐ │
│  │                    <body>                             │ │
│  │  ┌─────────────┐  ┌─────────────┐  ┌────────────────┐ │ │
│  │  │    <nav>    │  │   <main>    │  │   <footer>     │ │ │
│  │  │ Навигация   │  │   Контент   │  │   Подвал       │ │ │
│  │  └─────────────┘  └─────────────┘  └────────────────┘ │ │
│  └───────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

## Компоненты

### 1. Навигация (Navigation)
- **Логотип**: Ссылка на главную
- **Меню**: Переходы к секциям (Features, Pricing, Setup, FAQ)
- **Языковый переключатель**: 18 языков
- **Кнопки действий**: Канал и Бот

### 2. Секции контента

| Секция | ID | Описание |
|--------|-----|----------|
| Hero | - | Главный баннер с призывом к действию |
| Features | `#features` | 6 карточек возможностей |
| Setup | `#setup` | 4 шага подключения |
| Pricing | `#pricing` | 4 тарифа |
| Tech | - | Технологии и устройства |
| FAQ | `#faq` | Часто задаваемые вопросы |
| CTA | - | Финальный призыв |

### 3. Многоязычность (i18n)

```javascript
// Структура переводов
const base = {
  ru: { key: "Значение" },
  en: { key: "Value" },
  // ... другие языки
}
```

**Алгоритм работы:**
1. При загрузке определяется язык браузера
2. Применяются переводы из `base`
3. При смене языка в `<select>` - обновляется контент

## CSS Архитектура

### CSS Custom Properties (Переменные)

```css
:root {
  --bg: #f7f7f2;           /* Фон страницы */
  --surface: #ffffff;      /* Карточки */
  --ink: #11110f;          /* Основной текст */
  --muted: #696963;        /* Вторичный текст */
  --accent: #11110f;       /* Акцент */
  --green: #1f8a54;        /* Успех */
  --blue: #2867d8;         /* Информация */
  --radius: 8px;           /* Радиус скругления */
}
```

### Grid-система

- `.hero-grid` - Hero секция (2 колонки)
- `.cards` - Возможности (3 колонки → 2 → 1)
- `.steps` - Шаги (4 колонки → 2 → 1)
- `.pricing` - Тарифы (4 колонки → 2 → 1)
- `.faq` - FAQ (2 колонки → 1)

## JavaScript Модуль

### Основные функции

```javascript
// Инициализация
init()           // Запуск приложения
setLang(lang)    // Смена языка
t(key)           // Перевод по ключу
```

### События

- `DOMContentLoaded` - Инициализация
- `change` на `#langSelect` - Смена языка
