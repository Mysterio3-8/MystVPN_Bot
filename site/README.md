# MystVPN - Сайт лендинга

Современный лендинг для VPN-сервиса MystVPN с поддержкой многоязычности.

## Структура проекта

```
site/
├── index.html          # Главная страница
├── pages/              # Дополнительные страницы
│   ├── features.html   # Страница возможностей
│   ├── pricing.html    # Страница тарифов
│   ├── setup.html      # Страница подключения
│   └── faq.html        # Страница FAQ
├── css/
│   └── styles.css      # Основные стили
├── js/
│   └── app.js          # JavaScript приложения
└── README.md           # Документация
```

## Особенности

- **Многоязычность**: Поддержка 18 языков
- **Адаптивный дизайн**: Mobile-first подход
- **Telegram-интеграция**: Прямые ссылки на бота
- **Современный UI**: Минималистичный дизайн с акцентом на UX

## Технологии

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript
- i18n для локализации

## Запуск

Откройте `index.html` в браузере или разместите на веб-сервере.

## Разработка

### Добавление нового языка

1. Добавьте опцию в `<select id="langSelect">` в HTML
2. Добавьте переводы в объект `base` в `app.js`

### Добавление контента

1. Используйте атрибут `data-i18n="key_name"` для текста
2. Добавьте переводы для всех поддерживаемых языков
