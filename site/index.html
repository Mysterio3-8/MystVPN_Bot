WireGuard

Официальное приложение — работает для всех приложений

↓ Google Play

Шаги подключения

1

Получи ключ в боте

Открой бота @MysterioVPN_bot → Личный кабинет → нажми Инструкция. Бот пришлёт файл MystVPN.conf

2

Скачай WireGuard

Установи приложение WireGuard из Google Play (ссылка выше)

3

Импортируй файл

Открой WireGuard → нажми «+» → «Импорт из файлов» → выбери файл MystVPN.conf из Telegram

4

Подключись

Нажми переключатель рядом с MystVPN → разреши создание VPN → готово!

✅ Работает для всех приложений: Telegram, WhatsApp, TikTok, Instagram и любых других — без дополнительных настроек.

Шаги подключения

Получи ключ в боте

WireGuard для Windows

Официальный клиент от разработчиков WireGuard

↓ Скачать .exe

Шаги подключения

1

Получи ключ в боте

Открой бота @MysterioVPN_bot → Личный кабинет → нажми Инструкция. Бот пришлёт файл MystVPN.conf — сохрани его

2

Установи WireGuard

Скачай и запусти установщик по ссылке выше (требуются права администратора)

3

Импортируй туннель

Открой WireGuard → кнопка «Импортировать туннель(и) из файла» → выбери скачанный файл MystVPN.conf

4

Активируй

Нажми кнопку «Активировать» рядом с MystVPN → статус сменится на «Активен»

WireGuard для Mac

Официальное приложение из Mac App Store

↓ Mac App Store

Шаги подключения

1

Получи ключ в боте

Открой бота @MysterioVPN_bot → Личный кабинет → нажми Инструкция. Бот пришлёт файл MystVPN.conf

2

Установи WireGuard

Установи из Mac App Store по ссылке выше

Импортируй туннель

4

Нажми кнопку «Activate» → разреши добавление конфигурации VPN → готово!
