# CLAUDE.md — Traffic Bot (маркетинг / закуп каналов)

## Что это
Отдельный Telegram-бот для маркетинга и закупки рекламы в каналах. Раньше лежал в папке «Трафик-бот».

**Не путать с основным MystVPN ботом** (он в корне репозитория). Этот — про маркетинговые задачи.

## Статус
🟠 **В разработке.** Есть `bot.py`, `admin_panel.py`. Не в проде.

## Назначение
Автоматизация рутины:
- Поиск релевантных TG-каналов для закупки
- Парсинг метрик каналов (CPM, ER, аудитория)
- Отправка предложений админам каналов от моего лица
- Учёт закупов и их результатов (откуда сколько регистраций / оплат)
- Партнёрская программа: трекинг переходов от админов с %

## Технический стек
- Python + python-telegram-bot или aiogram (текущий выбор: см. `requirements.txt`)
- SQLite (`bot_data.db`) — для локальной разработки
- Свой `Dockerfile` + `docker-compose.yml` — изолирован от основного бота

## Файлы
- `bot.py` — основная логика
- `admin_panel.py` — админский интерфейс
- `config.py` — конфиг (токен, БД)
- `welcome_photo.jpg` — приветственное фото

## OPSEC
- Токен бота — только в `.env`, не коммитить
- При коммуникации с админами каналов — НЕ упоминаем напрямую что мы VPN-сервис
- Презентуем как «сервис свободного интернета», «доступ ко всем платформам»

## Запуск (локально)
```bash
cd traffic-bot
pip install -r requirements.txt
python bot.py
```

## Связи с MystVPN
- Может писать в общую БД основного бота (PostgreSQL) для трекинга UTM
- Использует тот же `services/payment_service.py` для верификации конверсий
