# Telegram-бот «Welcome-гейт»

Telegram-бот для управления доступом к приватному каналу с VPN-ключами. Пользователи должны подписаться на спонсорский канал, чтобы получить одноразовую ссылку-приглашение в закрытый канал.

## 🚀 Возможности

- ✅ Проверка подписки через Telegram API (`getChatMember`)
- 🔒 Генерация одноразовых ссылок-приглашений (срок действия 1 час)
- 📊 Админ-панель со статистикой
- 🗄️ База данных SQLite для хранения пользователей и ссылок
- 🖼️ Возможность обновления приветственного фото без перезапуска
- 📈 Отслеживание конверсии (переходы → подписки)

## 📋 Требования

- Python 3.10+
- Telegram Bot Token
- ID приватного канала
- Имя спонсорского канала

## 🛠️ Установка

1. Клонируйте репозиторий:
```bash
cd /path/to/project
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте конфигурацию в `config.py`:
```python
BOT_TOKEN = "ваш_токен_бота"
SPONSOR_CHANNEL = "имя_спонсорского_канала"  # без @
PRIVATE_CHANNEL_ID = -1001234567890  # ID приватного канала
```

4. Добавьте бота в администраторы:
   - В спонсорский канал (для проверки подписчиков)
   - В приватный канал (с правом создания ссылок-приглашений)

## 🚦 Запуск

```bash
python bot.py
```

## 🎮 Использование

### Для пользователей:

1. Нажмите `/start` для запуска бота
2. Подпишитесь на спонсорский канал
3. Нажмите «✅ Проверить подписку»
4. Получите ссылку в приватный канал

### Для администраторов:

- `/admin` — открыть админ-панель
- `/stats` — показать статистику
- `/setphoto` — обновить приветственное фото (отправьте фото после команды)
- `/settext` — обновить текст приветствия

## 🗄️ База данных

База данных SQLite (`bot_data.db`) содержит следующие таблицы:

- `users` — информация о пользователях
- `invite_links` — сгенерированные ссылки-приглашения
- `statistics` — ежедневная статистика

## 🔐 Безопасность

- Каждая ссылка уникальна и одноразовая (`member_limit=1`)
- Ссылки имеют срок действия (1 час)
- Проверка подписки через официальный Telegram API
- Защита от утечек (возможность отозвать все ссылки)
